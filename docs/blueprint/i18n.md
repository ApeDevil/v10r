# Internationalization Architecture

Locale-based routing with lazy-loaded translations using ICU MessageFormat.

---

## Strategy

**svelte-i18n** for per-language lazy loading that scales to many languages, backed by FormatJS.

| Aspect | Approach |
|--------|----------|
| Library | svelte-i18n |
| Routing | URL prefix (`/en`, `/de`) with param matcher |
| Translations | JSON files, lazy-loaded per locale + route |
| Message format | ICU MessageFormat (industry standard) |
| Date/Number | Native `Intl` API |
| Detection | URL → Cookie → Accept-Language |

### Why svelte-i18n

| Feature | svelte-i18n | Paraglide |
|---------|-------------|-----------|
| Per-language loading | Yes (via `register()`) | No (bundles all) |
| 10+ languages | ~14 KB + ~1 KB/locale | ~12+ KB × locales |
| Type safety | Partial | Full |
| Runtime switching | Yes | Requires reload |
| Message format | ICU (industry standard) | ICU |
| Maintenance | Active (v4.0.1, Oct 2024) | Active |
| Pluralization | CLDR-based, handles all languages | CLDR-based |

For 2-5 languages, Paraglide's type safety wins. For 10+ languages, svelte-i18n's lazy loading is essential.

**Why not sveltekit-i18n?** The library is unmaintained (seeking maintainers since 2023, last release July 2023) with no confirmed Svelte 5 support.

### Configuration

```typescript
// src/lib/i18n/index.ts
import { browser } from '$app/environment';
import { init, register, getLocaleFromNavigator } from 'svelte-i18n';

export const DEFAULT_LOCALE = 'en';
export const SUPPORTED_LOCALES = ['en', 'de', 'fr'] as const;
export type Locale = (typeof SUPPORTED_LOCALES)[number];

// Language display names
export const LOCALE_NAMES: Record<Locale, string> = {
  en: 'English',
  de: 'Deutsch',
  fr: 'Français',
};

// Route-to-namespace mapping for lazy loading
export const ROUTE_NAMESPACES: Record<string, string[]> = {
  '/': ['home'],
  '/about': ['about'],
  '/auth/login': ['auth'],
  '/auth/verify': ['auth'],  // OTP verification page (passwordless)
  '/dashboard': ['dashboard'],
  '/settings': ['settings'],
};

// Register all locales with lazy loading
for (const locale of SUPPORTED_LOCALES) {
  // Common namespace - always loaded
  register(locale, () => import(`./locales/${locale}/common.json`));
}

// Initialize svelte-i18n
init({
  fallbackLocale: DEFAULT_LOCALE,
  initialLocale: browser ? getLocaleFromNavigator() : DEFAULT_LOCALE,
});

// Re-export everything needed
export {
  _,
  locale,
  locales,
  isLoading,
  addMessages,
  waitLocale,
} from 'svelte-i18n';

// Alias for consistency
export { _ as t } from 'svelte-i18n';
```

### Namespace Loader

```typescript
// src/lib/i18n/loader.ts
import { addMessages } from 'svelte-i18n';
import { ROUTE_NAMESPACES, SUPPORTED_LOCALES, type Locale } from './index';

const loadedNamespaces = new Set<string>();

/**
 * Load translations for a specific locale and route.
 * Handles route-based lazy loading of translation namespaces.
 */
export async function loadTranslationsForRoute(
  locale: Locale,
  pathname: string
): Promise<void> {
  // Strip locale prefix from pathname
  const cleanPath = pathname.replace(/^\/(en|de|fr)/, '') || '/';

  // Find matching namespaces for this route
  const namespaces = findNamespacesForPath(cleanPath);

  // Load each namespace if not already loaded
  await Promise.all(
    namespaces.map((ns) => loadNamespace(locale, ns))
  );
}

function findNamespacesForPath(pathname: string): string[] {
  // Exact match first
  if (ROUTE_NAMESPACES[pathname]) {
    return ROUTE_NAMESPACES[pathname];
  }

  // Prefix match for nested routes
  for (const [route, namespaces] of Object.entries(ROUTE_NAMESPACES)) {
    if (pathname.startsWith(route) && route !== '/') {
      return namespaces;
    }
  }

  return [];
}

async function loadNamespace(locale: Locale, namespace: string): Promise<void> {
  const key = `${locale}:${namespace}`;

  if (loadedNamespaces.has(key)) {
    return;
  }

  try {
    const messages = await import(`./locales/${locale}/${namespace}.json`);
    addMessages(locale, { [namespace]: messages.default });
    loadedNamespaces.add(key);
  } catch (error) {
    console.warn(`Failed to load namespace ${namespace} for locale ${locale}`);
  }
}

/**
 * Reset loaded namespaces (useful for testing).
 */
export function resetLoadedNamespaces(): void {
  loadedNamespaces.clear();
}
```

### Translation Format (ICU MessageFormat)

```json
// src/lib/i18n/locales/en/common.json
{
  "nav": {
    "home": "Home",
    "about": "About",
    "contact": "Contact"
  },
  "button": {
    "save": "Save",
    "cancel": "Cancel",
    "delete": "Delete"
  },
  "greeting": "Hello, {name}!",
  "items": "{count, plural, one {# item} other {# items}}",
  "messages": "{count, plural, =0 {No messages} one {One message} other {# messages}}"
}
```

```json
// src/lib/i18n/locales/de/common.json
{
  "nav": {
    "home": "Startseite",
    "about": "Über uns",
    "contact": "Kontakt"
  },
  "button": {
    "save": "Speichern",
    "cancel": "Abbrechen",
    "delete": "Löschen"
  },
  "greeting": "Hallo, {name}!",
  "items": "{count, plural, one {# Element} other {# Elemente}}",
  "messages": "{count, plural, =0 {Keine Nachrichten} one {Eine Nachricht} other {# Nachrichten}}"
}
```

**ICU MessageFormat advantages:**
- Industry standard used by Java, iOS, Android
- Proper plural rules for ALL languages (Polish, Arabic, etc.)
- Translators already know it
- Supports select, date, time, number formatting inline

## Routing

### Param Matcher

Prevent route collisions by validating locale parameters:

```typescript
// src/params/lang.ts
import type { ParamMatcher } from '@sveltejs/kit';
import { SUPPORTED_LOCALES } from '$lib/i18n';

export const match: ParamMatcher = (param) => {
  // Only match non-default locales (English uses no prefix)
  return SUPPORTED_LOCALES.includes(param as any) && param !== 'en';
};
```

### Layout Server Load

```typescript
// src/routes/[[lang=lang]]/+layout.server.ts
import { loadTranslationsForRoute } from '$lib/i18n/loader';
import { DEFAULT_LOCALE, SUPPORTED_LOCALES, type Locale } from '$lib/i18n';
import type { LayoutServerLoad } from './$types';

export const load: LayoutServerLoad = async ({ params, url, cookies }) => {
  const locale = (params.lang as Locale) || DEFAULT_LOCALE;

  // Validate locale
  if (!SUPPORTED_LOCALES.includes(locale)) {
    return { locale: DEFAULT_LOCALE };
  }

  // Load translations for this route
  await loadTranslationsForRoute(locale, url.pathname);

  // Persist locale choice in cookie
  cookies.set('locale', locale, {
    path: '/',
    httpOnly: false,
    sameSite: 'lax',
    maxAge: 60 * 60 * 24 * 365, // 1 year
  });

  return { locale };
};
```

### Layout Client

```typescript
// src/routes/[[lang=lang]]/+layout.ts
import { browser } from '$app/environment';
import { locale, waitLocale } from '$lib/i18n';
import { loadTranslationsForRoute } from '$lib/i18n/loader';
import type { LayoutLoad } from './$types';

export const load: LayoutLoad = async ({ data, url }) => {
  const { locale: currentLocale } = data;

  // Set locale and load translations on client navigation
  if (browser) {
    locale.set(currentLocale);
    await loadTranslationsForRoute(currentLocale, url.pathname);
    await waitLocale(currentLocale);
  }

  return data;
};
```

### Route Structure

```
src/routes/
├── [[lang=lang]]/               # Optional locale prefix with matcher
│   ├── +layout.server.ts        # Server: load translations, set cookie
│   ├── +layout.ts               # Client: sync locale on navigation
│   ├── +layout.svelte           # SEO, hreflang tags
│   ├── +page.svelte             # Homepage
│   ├── about/
│   │   └── +page.svelte
│   └── dashboard/
│       └── +page.svelte
```

The `[[lang=lang]]` optional parameter with matcher means:
- `/` → English (default, no prefix)
- `/de` → German
- `/de/about` → German about page
- `/about` → Passes through (not a locale), renders English about page

## HTML Lang Attribute

**Important**: You cannot set the `<html lang>` attribute via `<svelte:head>`. Use hooks instead.

```html
<!-- src/app.html -->
<!doctype html>
<html lang="%lang%">
  <head>
    <meta charset="utf-8" />
    <meta name="viewport" content="width=device-width, initial-scale=1" />
    %sveltekit.head%
  </head>
  <body data-sveltekit-preload-data="hover">
    <div style="display: contents">%sveltekit.body%</div>
  </body>
</html>
```

```typescript
// src/hooks.server.ts
import type { Handle } from '@sveltejs/kit';
import { DEFAULT_LOCALE, SUPPORTED_LOCALES, type Locale } from '$lib/i18n';

export const handle: Handle = async ({ event, resolve }) => {
  // Get locale from URL params
  const lang = event.params.lang as Locale | undefined;
  const locale = lang && SUPPORTED_LOCALES.includes(lang) ? lang : DEFAULT_LOCALE;

  // Store in locals for use in load functions
  event.locals.locale = locale;

  return resolve(event, {
    transformPageChunk: ({ html }) => {
      return html.replace('%lang%', locale);
    },
  });
};
```

```typescript
// src/app.d.ts
declare global {
  namespace App {
    interface Locals {
      locale: import('$lib/i18n').Locale;
    }
  }
}

export {};
```

## Language Switcher

### With SvelteKit 2 Patterns

```svelte
<!-- src/lib/components/LanguageSwitcher.svelte -->
<script lang="ts">
  import { page } from '$app/state';
  import { locale, LOCALE_NAMES, SUPPORTED_LOCALES, DEFAULT_LOCALE } from '$lib/i18n';

  function getLocalizedPath(targetLocale: string): string {
    const pathname = page.url.pathname;
    const search = page.url.search;
    const hash = page.url.hash;

    // Build regex from supported locales
    const localeRegex = new RegExp(`^/(${SUPPORTED_LOCALES.join('|')})`);
    const pathWithoutLocale = pathname.replace(localeRegex, '') || '/';

    // Default locale uses no prefix
    const localizedPath = targetLocale === DEFAULT_LOCALE
      ? pathWithoutLocale
      : `/${targetLocale}${pathWithoutLocale}`;

    // Preserve query params and hash
    return localizedPath + search + hash;
  }
</script>

<nav aria-label="Language">
  {#each SUPPORTED_LOCALES as lang}
    <a
      href={getLocalizedPath(lang)}
      hreflang={lang}
      aria-current={lang === $locale ? 'page' : undefined}
    >
      {LOCALE_NAMES[lang]}
    </a>
  {/each}
</nav>
```

```svelte
<script lang="ts">
  import { page } from '$app/state';

  {#each SUPPORTED_LOCALES as lang}
```

```svelte
      rel="alternate"

    />

    rel="alternate"
```

## Date & Number Formatting

Use the native `Intl` API for locale-aware formatting.

### Formatters

```typescript
// src/lib/i18n/format.ts
import { locale } from '$lib/i18n';
import { get } from 'svelte/store';
import { DEFAULT_LOCALE } from './index';

/**
 * Get current locale with fallback.
 */
function getCurrentLocale(): string {
  return get(locale) || DEFAULT_LOCALE;
}

/**
 * Format a date according to current locale.
 */
export function formatDate(
  date: Date,
  options: Intl.DateTimeFormatOptions = { dateStyle: 'medium' }
): string {
  return new Intl.DateTimeFormat(getCurrentLocale(), options).format(date);
}

/**
 * Format date and time.
 */
export function formatDateTime(date: Date): string {
  return new Intl.DateTimeFormat(getCurrentLocale(), {
    dateStyle: 'medium',
    timeStyle: 'short',
  }).format(date);
}

/**
 * Format relative time (e.g., "2 days ago", "in 3 hours").
 */
export function formatRelative(date: Date): string {
  const rtf = new Intl.RelativeTimeFormat(getCurrentLocale(), { numeric: 'auto' });

  const diffMs = date.getTime() - Date.now();
  const diffMins = Math.round(diffMs / (1000 * 60));
  const diffHours = Math.round(diffMs / (1000 * 60 * 60));
  const diffDays = Math.round(diffMs / (1000 * 60 * 60 * 24));

  if (Math.abs(diffMins) < 60) {
    return rtf.format(diffMins, 'minute');
  }
  if (Math.abs(diffHours) < 24) {
    return rtf.format(diffHours, 'hour');
  }
  return rtf.format(diffDays, 'day');
}

/**
 * Format a number.
 */
export function formatNumber(
  value: number,
  options: Intl.NumberFormatOptions = {}
): string {
  return new Intl.NumberFormat(getCurrentLocale(), options).format(value);
}

/**
 * Format currency.
 */
export function formatCurrency(value: number, currency = 'USD'): string {
  return new Intl.NumberFormat(getCurrentLocale(), {
    style: 'currency',
    currency,
  }).format(value);
}

/**
 * Format percentage.
 */
export function formatPercent(value: number): string {
  return new Intl.NumberFormat(getCurrentLocale(), {
    style: 'percent',
    minimumFractionDigits: 0,
    maximumFractionDigits: 1,
  }).format(value);
}
```

### Usage

```svelte
<script>
  import { formatDate, formatCurrency } from '$lib/i18n/format';

  let { product } = $props();
</script>

<article>
  <h2>{product.name}</h2>
```

## Content Translation (Database)

For user-generated or CMS content stored in the database.

### Schema

```typescript
// src/lib/server/db/schema/content.ts
import { pgTable, text, jsonb, timestamp } from 'drizzle-orm/pg-core';

export const posts = pgTable('posts', {
  id: text('id').primaryKey(),
  slug: text('slug').notNull(),

  // Translated fields stored as JSON
  title: jsonb('title').$type<Record<string, string>>().notNull(),
  content: jsonb('content').$type<Record<string, string>>().notNull(),

  // Non-translated fields
  authorId: text('author_id').notNull(),
  publishedAt: timestamp('published_at'),
});

// Example data:
// title: { "en": "Hello World", "de": "Hallo Welt" }
```

### Content Helper

```typescript
// src/lib/i18n/content.ts
import { locale } from 'svelte-i18n';
import { get } from 'svelte/store';
import { DEFAULT_LOCALE } from './index';

type TranslatedField = Record<string, string> | null | undefined;

/**
 * Translate content from database JSON field.
 * Named `tc` (translate-content) to avoid collision with `t` from svelte-i18n.
 */
export function tc(
  translations: TranslatedField,
  fallback = DEFAULT_LOCALE
): string {
  if (!translations) return '';

  const currentLocale = get(locale) || DEFAULT_LOCALE;
  return (
    translations[currentLocale] ??
    translations[fallback] ??
    Object.values(translations)[0] ??
    ''
  );
}

/**
 * Check if translation exists for a locale.
 */
export function hasTranslation(
  translations: TranslatedField,
  targetLocale: string
): boolean {
  return Boolean(translations?.[targetLocale]);
}

/**
 * Get all available locales for a translated field.
 */
export function getAvailableLocales(translations: TranslatedField): string[] {
  if (!translations) return [];
  return Object.keys(translations).filter((key) => translations[key]);
}
```

### Usage

```svelte
<script>
  import { tc } from '$lib/i18n/content';

  let { post } = $props();
</script>

<article>
  <h1>{tc(post.title)}</h1>
  <div>{@html tc(post.content)}</div>
</article>
```

## Alternative: Paraglide

For projects with only 2-5 languages where type safety is the priority:

```typescript
// Paraglide usage (compile-time)
import * as m from '$lib/paraglide/messages';

m.greeting({ name: 'World' }); // Fully typed, autocomplete
```

Trade-offs:
- (+) Full type safety with autocomplete
- (+) Build-time validation of all keys
- (+) Smaller bundle for few languages
- (-) Bundles all languages together (issue #88 unsolved)
- (-) Runtime language switching requires reload

**Recommendation**: Use Paraglide for 2-5 languages, svelte-i18n for 10+.

## Migration from sveltekit-i18n

If migrating from sveltekit-i18n:

| sveltekit-i18n | svelte-i18n |
|----------------|-------------|
| `$t('key')` | `$t('key')` (same) |
| `{{name}}` | `{name}` |
| `{{count; 1:item; default:items}}` | `{count, plural, one {item} other {items}}` |
| `loadTranslations(locale, path)` | `loadTranslationsForRoute(locale, path)` |
| Built-in route matching | Manual via `ROUTE_NAMESPACES` |
| `$app/stores` | `$app/state` |
