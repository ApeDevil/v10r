# Cloudflare R2

Object storage. S3-compatible, zero egress fees, global CDN.

## Why R2

| Provider | Free Storage | Free Egress | Notes |
|----------|--------------|-------------|-------|
| **Cloudflare R2** | 10 GB | **Unlimited** | Best overall |
| Backblaze B2 | 10 GB | Via CDN only | Needs Cloudflare pairing |
| Supabase Storage | 1 GB | 2 GB/mo | Tied to Supabase |
| AWS S3 | 5 GB (12 mo) | 100 GB (12 mo) | Free tier expires |

R2 wins: zero egress, S3-compatible, built-in CDN, no credit card required.

See [../vendors.md](../vendors.md#cloudflare-r2) for pricing details.

## Stack Integration

| Layer | Technology | Why |
|-------|------------|-----|
| Protocol | **S3 API** | Standard, portable |
| Provider | **Cloudflare R2** | Zero egress, CDN |
| SDK | **@aws-sdk/client-s3** | Standard S3 API |
| Local Dev | **MinIO** | S3-compatible |

## Data Responsibilities

| Data Type | Store in R2 |
|-----------|-------------|
| User uploads | Yes |
| Images | Yes |
| Documents | Yes |
| Backups | Yes |
| User data | No (use PostgreSQL) |
| Relationships | No (use Neo4j) |

## Key Features

- **S3-compatible API** (same code works with AWS S3)
- **Zero egress fees** (serve directly or via CDN)
- **Presigned URLs** for secure direct uploads
- **Lifecycle rules** for automatic cleanup

## Local Development

Use MinIO container for local S3-compatible storage:

```yaml
# podman-compose.yml
minio:
  image: minio/minio
  ports:
    - "9000:9000"
  environment:
    MINIO_ROOT_USER: minioadmin
    MINIO_ROOT_PASSWORD: minioadmin
```

Same `@aws-sdk/client-s3` code works locally and in production.
