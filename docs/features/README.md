## Available Features

### Keyboard Shortcuts
Platform-aware keyboard shortcuts with sequence support (e.g., 'g h'). Includes help modal and full accessibility support.

**Key capabilities:**
- Single key combos (`mod+k`, `shift+?`)
- Multi-key sequences (`g h`)
- Platform detection (⌘ vs Ctrl)
- Context awareness (ignores inputs)
- Help modal with all shortcuts
